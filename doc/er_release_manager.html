get_recommended_versions/0

get_recommended_versions() -> {recommended_versions, [{{AppName, Stage}, Version}]}

Get the list of recommended versions for each stage of each app. Please note that recommendations may not always be available (for example when er_repository is down).

subscribe_recommended_versions/1

subscribe_recommended_versions(Subscriber::pid()) -> ok

Subscribe to version recommendations. Recommendations mean that it's safe to install given version of the AppName in given Stage immediately. The subsciber get the updates in the form of: {recommended_version,{{AppName,Stage},Number,mode()}}}
